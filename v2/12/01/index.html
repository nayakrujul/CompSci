GCSE Link: 4.06 - Hardware and Software

We know these two definitions from GCSE:

Hardware is the physical components of a computer system.

Software refers to the programs or instructions that control a computer's hardware.


Software can be broadly classified into two different groups:

Application software means programs which perform end-user tasks.

Within application software there are three main categories:

  • General-purpose: used for many different purposes, like word processors and web browsers
  • Special-purpose: for one specific task, like a music player
  • Bespoke: custom-built for an organisation or individual

System software means programs which are needed for the computer to function.

This consists of:

  • Operating Systems, which provide an interface between application software and the computer's hardware
  • Utility programs, which perform maintenance tasks like defragmentation and virus scanning
  • Libraries, which are collections of reusable code that other programs can use
  • Translators, which convert high-level code into low-level machine code that can be executed by the computer

We'll learn more about these types of system software over the next few pages.


What are the advantages and disadvantages of bespoke software over "off-the-shelf" (general- and special-purpose) software?

Advantages:
• Features are exactly as the customer requires, with nothing missing or unnecessary
• The client will have greater control over updates to the program
• Can provide an edge over competitors as they don't have access to the same software

Disadvantages:
• More expensive due to development costs
• Will not be available immediately
• Less likely to have as much support and maintenance as off-the-shelf software
