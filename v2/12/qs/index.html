Simplify the Boolean expression (A + B) · (C + D) + (A̅ · D) Use De Morgan's Laws on the first and last terms to get (A·B)·(C + D) + (A + D) Use the distributive property of AND over the first two terms to get (A·B·C) + (A·B·D) + A + D Simplify the last two terms using the absorption formula: (A·B·D) + D = (A·B) + D ; so the full expression is (A·B·C) + (A·B) + D + A Factorise the first two terms and simplify: (A·B·C) + (A·B) = (A·B)·(1 + C) =A·B ; so the full expression is A·B + D + A Apply the absorption formula again to the first and last terms, giving a final answer of A + B + D
