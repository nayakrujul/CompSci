In the A Level exam you may be asked to solve simple logic puzzles like this one:


A valuable jewel was taken from a museum by one of Alice, Bob or Carl. The following facts are gathered:

  1. Alice had an alibi after midnight, so she could only have stolen the jewel before midnight
  2. If and only if the thief used a rope, then Bob was in the storage room before the theft
  3. The jewel was stolen after midnight if and only if Carl was in the main hall before the theft
  4. Either Bob or Carl (but not both) was in the storage room before the theft
  5. If the thief used a crowbar, Bob is innocent
  6. Carl was either in the storage room or the main hall (but not both) before the theft
  7. The thief used either a crowbar or a rope (but not both)
  8. If Carl was not in the storage room before the theft then he is innocent
  9. If Carl was not in the main hall before the theft then Bob is the thief

A useful trick in these kind of logic puzzles is to use the contrapositive: the statement "if A then B" (which we can write as "A implies B") is logically equivalent to "if (not B) then (not A)" (or "B′ implies A′")


Who stole the jewel?

• We start with clue 7 which tells us that either a crowbar or rope was used, but not both. We examine the cases separately:

Crowbar:
• From clue 5, if a crowbar was used then Bob is innocent
• Clue 2 tells us that Bob was not in the storage room before the theft in this case
• Using clues 4 and 6, this means that if a crowbar was used then Carl was in the storage room before the theft
• But from clue 9, Bob is the thief: we have reached a contradiction, therefore the thief did not use a crowbar

Rope:
• Using clue 2, if a rope was used then Bob was in the storage room before the theft
• From clues 4 and 6, if the thief used a rope then Carl was in the main hall (and not in the storage room)
• But from clue 8 this means that Carl is innocent
• Clue 3 tells us that this means the jewel was stolen after midnight
• Therefore from clue 1, Alice is innocent, and so Bob is the thief
