There are two main types of designs of computer systems.

In the von Neumann architecture, data and program instructions are stored together in a single pool of main memory.

In the Harvard architecture, data and instructions are stored separately in two distinct memory spaces.

The von Neumann architecture is used in all modern general-purpose devices due to its flexibility, while embedded systems might use the Harvard architecture due to its ability to access data and instructions in parallel. The Harvard architecture can also prevent data from being executed as code (an exploit that could be used by hackers).


The stored program concept is the idea that instructions can be stored in and fetched from main memory, just like data. This allows for different programs to be run without changing the hardware.


How does the stored program concept relate to the von Neumann architecture?

The stored program concept is the basis of the von Neumann architecture, where data and program instructions are stored together in main memory and transmitted along the same buses.
