GCSE Link: 4.09 - The CPU


Processors consist of a number of components:

  • The Control Unit (CU) coordinates the activities of the CPU, identifying the processes needed to carry out each instruction and sending control signals to the appropriate components.
  • The Arithmetic and Logic Unit (ALU) performs mathematical (addition/subtraction), logical (AND, OR, XOR, NOT) and bit-shift operations.
  • The clock which generates an oscillating signal to synchronise the activity of CPU components.
  • General-purpose registers are small units of memory in the CPU that can store anything, usually an intermediate result of a calculation or a memory address.
  • Dedicated registers (or special-purpose registers) are used to hold specific values needed by the CPU. They include:
     
    • The Program Counter (PC), which holds the memory address of the next instruction to be executed. It is updated at the end of the Fetch stage of the Fetch-Decode-Execute cycle.
    • The Current Instruction Register (CIR), which holds the current instruction which is to be decoded and executed.
    • The Memory Address Register (MAR), which holds the memory address of data which is to be read from or written to main memory.
    • The Memory Buffer Register (MBR), which holds the actual data which has been read from or is to be written to main memory.
    • The Status Register (SR), which holds status codes relating to the executed instruction (e.g. if an overflow has occurred during a calculation).

Some processors can also include some cache memory, which is very fast memory inside the CPU. It stores recently and frequently used instructions and data so that they don't need to be fetched from main memory if needed again.


How could CPU performance be affected by amount of cache memory? What about other factors?

• More cache memory means more instructions and data can be retrieved faster, which improves CPU performance
• Having multiple CPU cores can also improve performance if tasks can be performed in parallel
• Increasing the clock speed will mean more Fetch-Execute cycles in a second, which improves performance
• A higher word size (the number of bits the CPU can process at once) will also improve CPU performance
• Increasing the width of the address bus will also improve performance, as more memory locations can be addressed
• Similarly, increasing the width of the data bus will also improve performance, as more data can be sent at once
