All computer systems require certain essential components.

The processor or Central Processing Unit (CPU) executes machine code instructions to run programs.

This will be explored further later on in this section.


Main memory stores data and program instructions so that they can be fetched by the processor.

Main memory is split into small blocks (usually bytes), each of which has an address that can be used to retrieve the data in that space.


Buses are collections of wires used to send data between components of the computer system.

The three main buses are:

  • The data bus, a bidirectional bus which carries data between the CPU and other components such as main memory
  • The address bus, a unidirectional bus which transfers memory addresses from the CPU to other components
  • The control bus, a bidirectional bus carrying signals such as the clock and read/write mode

The width of a bus is the number of wires that it has. Since each wire can transmit a single bit of information at once, the total number of memory locations that can be transmitted by the address bus is 2w, where w is the width of the bus.


Input/Output controllers are hardware components that manage communication between the processor and external devices.

They include a port to connect to the peripheral device, a set of registers to hold data, and an interface to connect to the processor.


What is the minimum required address bus width to address 16GB of RAM, where each memory location is a byte?

16GB is equal to 16,000,000,000 bytes. Taking log2 of this gives 33.9, so the minimum width required is 34.
