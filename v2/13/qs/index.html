Explain how each of the three buses are used when writing to main memory. The memory address to be written to is placed on the address bus The data to be written is placed on the data bus The write signal is placed on the control bus Then a clock signal is carried by the control bus While the write signal is received by the memory, the data from the data bus is stored in the address specified by the address bus
State two differences between the Harvard and von Neumann architectures. In the von Neumann architecture, instructions and data are stored in the same memory and are transmitted using the same buses In the Harvard architecture, instructions and data can be fetched simultaneously
A computer's address bus has a width of 32 bits, and each memory location holds a 16-bit data value. Calculate the maximum amount of main memory that could be installed in the computer in gibibytes. First mark for the calculation of 232 × 16 ÷ 8 = 233 bytes Second mark for the final answer of 233 ÷ 230 = 8 gibibytes
Name four special-purpose registers, and describe their purpose Program Counter: holds the memory address of the next instruction to be fetched and executed Current Instruction Register: holds the current instruction which is to be decoded Memory Address Register: holds the memory address of data which is to be read from or written to main memory Memory Buffer Register: holds the data which has been read from or is to be written to main memory Status Register: holds status codes relating to the executed instruction
What is the stored program concept? Instructions are treated as data and stored in main memory Programs can be easily loaded, changed or executed in any order
Describe and explain three steps that a processor goes through during the fetch stage of the Fetch-Execute cycle. The contents of the Program Counter are copied onto the Memory Address Register so that the next instruction can be fetched from main memory The contents of the Memory Address Register are placed onto the address bus so that the correct memory location can be accessed The value received on the data bus is loaded into the Memory Buffer Register because it cannot be directly loaded into the Current Instruction Register The contents of the Memory Buffer Register are copied into the Current Instruction Register to allow other data to be fetched during the execute phase The Program Counter is incremented so that it points to the next instruction to be fetched
