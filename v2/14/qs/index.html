Calculate the bit rate of a communication that has a Baud rate of 5 MegaBaud and 4 possible voltage levels. 4 voltage levels means 2 bits per signal So bit rate is 10Mbps
What is the difference between asynchronous and synchronous communication? In synchronous transmission, the receiver and transmitter are (continuously) synchronised by a shared clock
What is the difference between serial and parallel transmission? Serial uses a single wire/cable, while parallel uses multiple Serial sends one bit at a time, while parallel sends multiple bits simultaneously
Explain two disadvantages of parallel transmission. Extra wiring is required which makes parallel transmission more expensive than serial transmission Crosstalk (interference) between wires can be an issue over longer distances making parallel transmission less reliable than serial Imperfections in the wires could mean that data isn't received at the same time (data skew) causing a delay at the receiving end, slowing down the transmission
State the purpose of the start and stop bits in asynchronous communication. Start bit used to start the receiver's clock and synchronise it with the transmitter's clock Stop bit used to provide time for the receiver to process the data, and to allow the next start bit to be recognised
Describe the operation of the CSMA/CA protocol with RTS/CTS. The device listens to the network to check if it is idle before attempting to transmit If the network is busy, it waits for a random time period before trying again When the network is free, the device transmits a Request to Send frame The receiver then responds with a Clear to Send After hearing this, all other devices on the network go silent for some time After data has been transmitted, the receiver sends an acknowledgement This notifies the other devices that they can transmit again If the sender did not receive an acknowledgement, it waits and tries again
