Computer A wants to send a message to Computer B using asymmetric encryption. A digital signature will also be sent to verify that the message was sent by Computer A. Describe the entire process, of encryption and decryption. The message is hashed and this hash is encrypted using A's private key The encrypted hash (known as the digital signature) is appended to the message The full message (along with the signature) is encrypted using B's public key B's private key is used to decrypt the full message (and digital signature) A's public key is used to decrypt the encrypted hash The hash is recalculated from just the message If they match, B knows that the message was sent by A
