GCSE Link: 6.04 - Malware

Malware is an umbrella term used to refer to a variety of forms of hostile or intrusive software.

There are many different forms of malware

A virus is a type of malware that is attached to another program or file, called a host.

Viruses can be memory resident (they can stay in memory even after the host has been closed) or non-memory resident (they only execute while the host is open).

A worm is a type of malware that can replicate without user interaction.

Worms often spread throughout a network by themselves, damaging or stealing data.

A Trojan refers to malware disguised as legitimate software.

They take their name from the story of the Trojan War, where Greek soldiers hid inside a wooden horse to enter the city of Troy.


Attackers can exploit poor code quality in a system (for example, utilising buffer overflows or code injections). Good input validation and regular code testing can help protect against malware.

Another method used to protect against malware is monitoring and protection. Software such as antivirus programs can scan for known malware and remove them before they cause damage. Regularly updating software can also close security holes that could be exploited by malware.


What are some other forms of malware?

• Ransomware, which can make files unusable until the user pays the hacker
• Spyware, which can track a user's activity
• Adware, which can display unwanted advertisements
• Botnets, which can use a user's device in a DDoS attack
