4.05 – Logic Symbols

Logic symbols can also be used to represent boolean expressions.

Table 1 shows the corresponding symbol for each logic gate.

Table 1

Logic gate Symbol
NOT(A) A
A AND B A . B
A OR B A + B
A XOR B A ⊕ B


Note: the bar needs to go over everything inside the NOT: for example, NOT(A AND B) would become (A . B)
