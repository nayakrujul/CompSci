7.02 – Select Statements

Computer Science Revision


Previous: 7.01 - Databases

Structured Query Language (SQL, pronounced "sequel") provides an efficient way to retrieve information from a database.

The most basic SQL command is the SELECT statement, which can retrieve data from a database.

Example 1 shows the SQL statement to SELECT all date of births from Students.

Example 1
SELECT DateOfBirth FROM Students

To select multiple fields, use commas to separate them.

Example 2 shows the SQL statement to SELECT all first names and date of births from Students.

Example 2
SELECT FirstName, DateOfBirth FROM Students

Finally, use * to SELECT all the fields.
